All Commands

Prefix = /

ban - Bans a user from the server
kick - Kicks a user from the server
lock - Locks a specified channel
role - Gives a user the specified role
slowmode - Sets the slowmode of a channel
unlock - Unlocks a specified channel
unmute - Unmutes a user
mute - Mutes a user
warn - Warns a user
clear - clears a specified amount of messages

8ball - Ask a question to the magic 8Ball.
asciiArt - Displays an ascii art image.
avataravatar - Displays the avatar of a user.
help - Shows all the commands
meme - Shows a meme from reddit
serverinfo - Shows information about a server
userinfo - Shows information about a user

ticket - Adds a member to a ticket
ticket-setup - Sets up the ticket system
voice - Control your private voice channel
afk - Sets your status to afk
